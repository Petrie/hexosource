很久就想写一篇关于shell的for语法的博客了。预谋了很久，今天终于提笔
核心语法
``` bash
for i in `ls`;do echo $i;done;
```
常见场景：
>查询一组host对应的IP

for计数和shell的四则运算
``` bash
for((i=1;i<=10;i++));do echo $(expr $i \* 4);done
```

for计数2
``` bash
for i in $(seq 10)
```

for循环命令执行结果
``` bash
for i in `ls`
```
或者
``` bash
for i in $(ls)
```

for循环数组变量
``` bash
arr=(1 2 3 4);for i in ${arr[@]};do echo $i;done;
```
